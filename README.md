# 遊戲玩法

Jett有一天發現風景照都缺了一塊！
 
請觀察Target圖中所缺失的部分，並切換到Quiz模式找出缺失的部分
        
- 正確：獲得兩分

- 錯誤：倒扣一分
        
請在六十秒內努力的將照片們都恢復原狀吧！
